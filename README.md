# AuraSec AI - Unauthorized Data Access Monitoring

![AuraSec AI Hero](home%20page.png)

AuraSec AI is an advanced, automated network monitoring system designed to detect and resolve unauthorized data access attempts in real-time. Built with a stunning "liquid glass" interface and powered by AI, it offers seamless functionality across desktop and mobile platforms.

## 🌟 Key Features

* **AI-Automated Security Assistant**: A built-in AI chatbot capable of analyzing network anomalies, providing detailed system status reports, and executing security protocols.
* **1-Button "Magic Fix"**: An automated workflow that instantly patches vulnerabilities, blocks unauthorized IP addresses, and secures breached network nodes with a single click.
* **Live Threat Topology Map**: Visualizes the network architecture in real-time. Breached nodes are highlighted and pulsating, giving administrators immediate visual feedback on active threats.
* **Liquid Glass UI**: Features a modern, premium "glassmorphism" aesthetic with a light blue gradient theme, complete with micro-animations and blur effects.
* **Responsive Architecture**: The web application dynamically morphs into a native mobile app layout on smaller screens, complete with a dedicated bottom navigation bar for seamless cross-platform usage.
* **Native Mobile Capabilities**: Wrapped with Capacitor, the application is ready to be built and deployed natively for Android and iOS.

## 🚀 Getting Started

### Prerequisites
* Node.js (v16 or higher)
* npm or yarn

### Installation
1. Clone the repository
   ```bash
   git clone https://github.com/yourusername/AuraSec-AI.git
   ```
2. Navigate to the project directory
   ```bash
   cd "AuraSec-AI"
   ```
3. Install dependencies
   ```bash
   npm install
   ```

### Running the Web Application
```bash
npm run dev
```
Navigate to `http://localhost:5173/` in your browser.

### Running the Mobile App
The project is configured with Capacitor for native mobile deployment.
```bash
npm run build
npx cap sync
npx cap open android # Requires Android Studio
```

## 🛠️ Technology Stack
* **Frontend**: React.js, Vite
* **Styling**: Vanilla CSS (Glassmorphism design system)
* **Icons**: Lucide React
* **Animations**: Framer Motion
* **Mobile Wrap**: Capacitor JS
